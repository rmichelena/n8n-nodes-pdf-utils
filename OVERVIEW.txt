│  Custom node para n8n que permite:                                          │
│  • Inspeccionar PDFs (páginas, texto, tipo)                                 │
│  • Dividir PDFs multipágina en archivos individuales                        │

│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────────────────────┐
│ ⚡ OPERACIONES                                                               │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  1️⃣  INSPECT                                                                │
│     ┌────────────┐      ┌────────────┐      ┌──────────────────┐           │
│     │ PDF binary │  →   │ PDF Utils  │  →   │ {                │           │
│     │            │      │ (Inspect)  │      │   pageCount: 5,  │           │
│     └────────────┘      └────────────┘      │   isVectorial: T,│           │
│                                              │   textLength: 99 │           │
│                                              │ }                │           │
│                                              └──────────────────┘           │
│     ⏱️  Performance: ~10-50ms                                               │
│                                                                              │
│  2️⃣  SPLIT                                                                  │
│     ┌────────────┐      ┌────────────┐      ┌──────────────────┐           │
│     │ PDF 5 págs │  →   │ PDF Utils  │  →   │ 5 items:         │           │
│     │            │      │  (Split)   │      │ • page_1.pdf     │           │
│     └────────────┘      └────────────┘      │ • page_2.pdf     │           │
│                                              │ • page_3.pdf     │           │
│                                              │ • page_4.pdf     │           │
│                                              │ • page_5.pdf     │           │
│                                              └──────────────────┘           │
│     ⏱️  Performance: ~50-200ms por página                                   │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────────────────────┐
│ 🎓 DECISIONES TÉCNICAS CLAVE                                                │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  ❓ ¿Por qué NO canvas?                                                     │
│     Canvas solo se necesita para renderizar PDFs como imágenes.            │
│     Nosotros solo extraemos texto y metadata → NO lo necesitamos.          │
│                                                                              │
│  ❓ ¿Por qué pdfjs-dist en modo headless?                                   │
│     Configuramos pdfjs-dist para funcionar sin DOM/canvas:                 │
│     • pdfjsLib.GlobalWorkerOptions.workerSrc = ''                           │
│     • useWorkerFetch: false                                                 │
│     • isEvalSupported: false                                                │
│                                                                              │
│  ❓ ¿Por qué estas librerías específicas?                                   │
│     • pdfjs-dist: Battle-tested (usado en Firefox)                          │
│     • pdf-lib: Puro JavaScript, sin deps nativas                            │
│     • Ambas mantenidas activamente                                          │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────────────────────┐
│ 💼 CASOS DE USO                                                             │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  1. Clasificar PDFs automáticamente                                         │
│     Webhook → Inspect → IF vectorial → Proceso A                           │
│                                    ELSE → OCR (Proceso B)                   │
│                                                                              │
│  2. Procesar PDFs página por página                                         │
│     HTTP Request → Split → Loop → Procesar individualmente                  │
│                                                                              │
│  3. Pipeline de documentos                                                  │
│     Drive Trigger → Inspect → IF multipage → Split → Procesar              │
│                                                                              │
